Hardware y Software

¿Qué es el hardware?

El hardware es el conjunto de componentes físicos (tangibles) que forman parte de una computadora. Incluye dispositivos como el monitor, teclado, disco duro y placa madre.

Clasificación

Generalmente el hardware se clasifica en dispositivos o Periféricos de Entrada y Salida. Entendiendo por dispositivo a todo aquel que se conecta a una PC para optimizar su funcionamiento o complementarlo.

PERIFÉRICOS O DISPOSITIVOS DE ENTRADA: Son los que permiten que el usuario ingrese información a la computadora.

PERIFÉRICOS O DISPOSITIVOS DE SALIDA: Son los que permiten sacar información o muestran al usuario el resultado de las operaciones realizadas por el PC.

PERIFÉRICOS O DISPOSITIVOS DE ENTRADA/SALIDA: Son los dispositivos que pueden cumplir ambas funciones, o sea que permiten al usuario ingresar y sacar información de la computadora.

¿Qué es el software?

El software de una computadora es todo aquel que le permite al usuario ordenarle a la misma que realice una tarea y permite que el hardware funcione. Es la parte no física (intangibles) de una computadora, básicamente está compuesto por todos los programas y el sistema operativo.

Clasificación

El software suele clasificarse en:

SOFTWARE DE APLICACION:Son los programas que permiten al usuario realizar tareas específicas, como procesadores de texto, hojas de cálculo, navegadores web, etc.

SOFTWARE DE SISTEMA: Es el software que gestiona los recursos de hardware y proporciona una plataforma para que se ejecuten otros programas. El sistema operativo es un ejemplo de software de sistema.

SOFTWARE DE PROGRAMACION: Son herramientas que permiten a los desarrolladores crear, depurar y mantener software. Incluyen lenguajes de programación, compiladores, editores de código, etc.
